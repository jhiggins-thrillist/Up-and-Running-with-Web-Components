Workshop #1 - Native Web Components

Welcome to our first workshop! We will now be implementing features of our web component, the shiny & new <tmg-button>.

Tasks:

  • Modify the imported tmg-button.html so it generates HTML content from it's template. +1 point
  • The textContent of <span id="#title"> in the custom element matches the title attribute of <tmg-button title="">. (1 point)
  • When the button is clicked, it should will alert its content attribute. (1 point)

Deductions:

  • Using jQuery (-3 points)

Extra Credit:

  • Add Object.Observe() functionality to the <tmg-button>. (2 points)
    • When the title attribute is changed in the Inspector, make sure it changes in the Shadow DOM.
  • Have the select dropdown change the button's title text by using a MutationObserver. (2 points)
